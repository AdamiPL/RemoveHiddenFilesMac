# RemoveHiddenFilesMac

How install?
Download this script and give him chmod a+x then you can use from your terminal.

chmod a+x ~/Desktop/clean.sh

How to use?

./clean.sh [path]
./clean.sh /Volumes/Windows/ 

Finding and deleting:

.DS_Store
.Trashes
.fseventsd
.TemporaryItems
.apdisk
._* (dot underscore files)
